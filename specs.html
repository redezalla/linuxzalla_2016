LINUX

por Rafael Sepulveda Zalla

2017

A filosofia do Linux é sintetizada pela facilidade com que se pode abrir um emulador de terminal para acessar o CLI e seu poder concomitante. Primeiro, existem os vários terminais virtuais que podem ser acessados ??usando as teclas Ctrl-Alt-F [1-7]. Mesmo os desktops Linux GUI sussurros, "Use a força, Luke," para todos os que usá-los. O Linux tem vários ambientes de desktop GUI finos de onde escolher, para que cada usuário possa escolher o que mais lhe agrada. E ainda, o poder da CLI é apenas um clique de distância com muitas opções para o emulador de terminal eo shell. Meu emulador de terminal favorito para o desktop GUI é o Konsole. O Konsole permite várias sessões de terminal em uma janela GUI usando guias para alternar entre sessões de terminal. Claro que você pode usar seu shell favorito em todas as sessões ou um shell diferente em cada sessão, dependendo de suas necessidades. Para ainda mais flexibilidade, existe o programa de tela que fornece várias sessões de shell dentro de uma única sessão de terminal. Isso é particularmente útil ao fazer login remotamente porque, se sua conexão falhar, a sessão de tela continuará junto com todos os programas em execução nos vários shells lançados na sessão de tela. O Linux CLI seduz suavemente; Ele atormenta com seu poder. É a flexibilidade que me atraiu. Isso me mudou - certamente mudou a maneira como eu uso o computador. Eu ainda uso a área de trabalho GUI, mas tanto para me dar a flexibilidade de várias sessões do Konsole com tela em execução para executar programas como LibreOffice, Firefox e Thunderbird. todos os sistemas operacionais têm uma filosofia. E, a filosofia de um sistema operacional importa. Qual é a filosofia do Linux e como isso afeta a comunidade? Como ele mudou o desenvolvimento de software para as idades? Se sabemos ou não, a maioria de nós tem algum tipo de filosofia de vida. Pode ser tão simples quanto: "Seja gentil com os outros", ou pode ser uma filosofia de vida muito complexa. Antes de começar a trabalhar com o Linux em alguma profundidade, eu nunca tinha sabido que tinha uma filosofia. Quero dizer, o que uma filosofia poderia realmente fazer para um sistema operacional? Depois de um pouco de pesquisa, descobri que todos os sistemas operacionais têm uma filosofia. Aprendi também que a filosofia de um sistema operacional é importante. Por exemplo, a filosofia do Windows é muito semelhante à da Digital Equipment Corporation (DEC) VMS sistema operacional. Isso equivale a "proteger os usuários de tudo o que possa levá-los a problemas." Claro, a razão para essa semelhança é que o desenvolvedor principal do Windows NT é a mesma pessoa que desenvolveu o VMS. A filosofia de ambos os sistemas operacionais é baseada na crença subjacente de que os usuários têm medo de computadores e precisam ser protegidos de sua complexidade. O MAC OS não é muito diferente, embora a interface gráfica do usuário (GUI) desktop tem uma abordagem orientada a objeto que, na opinião de muitos, fornece uma experiência de usuário mais consistente, integrado e muito mais suave. Ambos MAC e Windows fornecem uma interface de linha de comando (CLI) que permite interação de baixo nível entre o usuário eo sistema operacional. No entanto, a CLI do Windows é muito limitada com poucos comandos, pelo menos quando comparado ao Linux e Unix. O MAC CLI é um shell BASH que fornece uma grande quantidade de energia, assim como o faz no Linux. No entanto, é muito difícil de encontrar. O shell é enterrado em uma avalanche de opções de menu que só pode ser encontrado se você sabe que ele está lá e estão ativamente procurando por ele. O Linux não controla. Ele assume que você sabe o que você está fazendo quando você digita um comando e ele continua a executar esse comando sem perguntar se você realmente quer. Dá-lhe o controle completo. Imagine por um momento o caos ea frustração que resultaria da tentativa de usar uma pistola de pregos que lhe perguntou se você realmente queria atirar nessa unha e não iria permitir que você puxe o gatilho até que você disse a palavra "sim" em voz alta. O Linux permite que você use a pistola de unhas como você escolher. Outros sistemas operacionais permitem que você saiba que você pode usar pregos, mas não dizer-lhe que ferramenta é usada para inserir os pregos e muito menos permitem que você coloque o seu próprio dedo sobre o gatilho. Sim, há perigo onde há grande poder. Usado sabiamente que o poder também pode ser aproveitado para realizar muitas grandes coisas.
